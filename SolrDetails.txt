Solr Details on local - 
youtube video - https://www.youtube.com/watch?v=v28FJswU1DQ

Extracted dir - D:\softwares\solr-7.0.1
To start - Go to cmd prompt - location D:\softwares\solr-7.0.1\bin
Run - solr start 
OR solr start -p 8983 (default port is 8983; you can change)

Solr home dir - D:\softwares\solr-7.0.1\server\solr - as mentioned in youtube video above
It will create separate dirs for each core you create.

To know how to index - refer this short video (6 mins) - https://www.youtube.com/watch?v=rxoS1p1TaFY&pbjreload=10
To create new core - Run below command -
D:\softwares\solr-7.0.1\bin>solr create -c dance
Response -

         To turn it off:
            curl http://localhost:8983/solr/dance/config -d '{"set-user-property": {"update.autoCreateFields":"false"}}'

D:\softwares\solr-7.0.1\server\solr\dance
